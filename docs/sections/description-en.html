ATM Information Reference Model Ontology: Description back to ToC

An OWL representation of the AIRM – the AIRM Ontology (AIRM-O) – facilitates handling in practice. AIRM-O is a semi-automatic derivation of AIRM and as such closely follows the structure and semantics of the UML models. We hence refer to the AIRM website for general information on meaning and usage of the model elements themselves. In addition to that, we provide support material (see references) that illustrates potential applications of AIRM-O in practice and describes how to work with AIRM-O.
